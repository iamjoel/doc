# Shell
## 运行 Shell 文件

```bash
shell 文件名
```

## 查看进程

```bash
ps
ps|grep 'yarn start'
```

## 杀死进程

```bash
kill 进程id
```
## 使用 nvm

```bash
export NVM_DIR=$HOME/.nvm
source $NVM_DIR/nvm.sh

nvm use v14
node -v
cd '...'
yarn start
```
